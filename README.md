# Employee Management System

A full-stack web application built from scratch.

## Tech Stack
- **Backend:** Java 17 + Spring Boot 3 + Spring Data JPA + H2 Database
- **Frontend:** React 18 + Axios

## Project Structure
- `/backend` → Spring Boot REST API
- `/frontend` → React UI

## Getting Started
### Backend
```bash
cd backend
./mvnw spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm start
